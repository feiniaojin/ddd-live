# DDD-LIVE

DDD-LIVE是基于DDD实现的直播服务。本项目是作者的新书《悟道领域驱动设计》的随书案例代码。《悟道领域驱动设计》目前已上架各大电商平台，感兴趣的朋友可以帮忙支持一下。

<img alt="悟道领域驱动设计" src="assets/book.png" style="width: 50%">

购买链接：https://item.jd.com/14835288.html

配套前端工程地址：

```text
https://github.com/feiniaojin/ddd-live-front
```

# 技术栈

- SpringBoot
- MyBatis、Spring Data JDBC
- Graceful-Response
```text
https://github.com/feiniaojin/graceful-response
```
- ddd-archetype
```text
https://github.com/feiniaojin/ddd-archetype
```
- MySQL
- 阿里云视频直播服务
